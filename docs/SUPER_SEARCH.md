# 🔎 Super Search (Ctrl + Q)

## Overview
Super Search is a powerful, context-aware search tool for Diagflo. Instantly find documentation, components, commands, and more—across the entire workspace.

## How to Use
- **Open Super Search:** Press <kbd>Ctrl</kbd> + <kbd>Q</kbd> anywhere in the app.
- **Type your query:** Search for files, components, docs, or keywords.
- **Navigate results:** Use arrow keys to move, <kbd>Enter</kbd> to open.

## Features
- Fuzzy matching for fast results
- Previews for docs and code
- Keyboard navigation
- Search across docs, code, and UI

## Tips
- Search “button” to find all button components and docs.
- Type “design” to jump to the Design System.
- Use “api” to find backend endpoints.
