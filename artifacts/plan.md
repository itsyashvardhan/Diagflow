# Plan: Jony Ive Design Refinement for Diagflow

Apply Sir Jony Ive's principles of minimalism, precision, and high-quality materials to the Diagflow app without losing its "warm orange" vibe.

## 1. Material Refinement (CSS)
*   **Perfect Glass**: Enhance the `.glass-panel` utility with multi-layered borders and subtle internal highlights to simulate physical glass.
*   **Premium Blurs**: Increase backdrop blur for a more professional, "frosted" look.
*   **Typography**: Refine font weights and letter-spacing for high-end readability.

## 2. Layout Precision (Index.tsx)
*   **Minimalist Header**: Simplify the header into a unified toolbar. Reduce visual clutter from ghost button borders.
*   **Elegant Splitter**: Redesign the resize handle to be a minimal, high-precision tactile strip.
*   **Refined Empty State**: Polish the welcome screen with better staggered animations and a cleaner structure.

## 3. UI Control Refinement
*   **Floating Toolbars**: Refine `DiagramControls` and `ZoomControls` to feel more like high-end physical hardware interfaces.
*   **Subtle Interactions**: Add hover states that use luminosity and scale rather than just color changes.

## 4. Components Cleanup
*   Ensure all modals share the new premium glass style.
*   Refine the chat input area to be more integrated.
