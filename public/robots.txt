# Diagflo
#
# Diagflo is an AI-powered diagram generation tool. Describe your architecture,
# system design, or workflow in plain language and get a professional diagram
# in seconds.
#
# Website: https://diagflo.yashvardhan.dev
#
# Features:
# - AI-Powered Generation (Mermaid.js + Chart.js)
# - Supported: Flowcharts, sequence, class, ER, state, Gantt, pie, mind maps,
#   Git graphs, scatter/line/bar with log scales and annotations
# - Image-to-Diagram: Upload whiteboard sketches → clean editable diagrams
# - Iterative Refinement: Chat with Archie to refine with follow-up prompts
# - Export: SVG, PNG, or raw Mermaid/Chart.js code
# - Shareable Links: Unique URL to share any diagram, no login required
# - Privacy First: API key stays in browser, diagrams generated client-side
# - Dark and Light Theme
#
# Tech Stack: React, TypeScript, Vite, Tailwind CSS, Mermaid.js, Chart.js,
#             Google Gemini API, Vercel, Supabase
